OpenAITranslator-AI翻译

OpenAITranslator是一个使用 ChatGPT API 进行划词翻译、总结、润色、分析、代码解释的开源浏览器插件和桌面软件。借助了 ChatGPT 强大的翻译能力，它将帮助您更流畅地阅读、编辑外语。


特点

  1. 支持三种翻译模式：翻译、润色、总结
  2. 支持 55 种语言的相互翻译、润色和总结功能
  3. 支持实时翻译、润色和总结，以最快的速度响应用户，让翻译、润色和总结的过程达到前所未有的流畅和顺滑
  4. 支持自定义翻译文本
  5. 支持一键复制
  6. 支持 TTS
  7. 有桌面端应用，全平台（Windows + macOS + Linux）支持！
  8. 支持截图翻译
  9. 支持生词本，同时支持基于生词本里的单词生成帮助记忆的内容
  10. 同时支持 OpenAI 和 Azure OpenAI Service
